== Installation ==  
1. Upload the `brainfwd` theme folder to `/wp-content/themes/` or install via WordPress theme uploader.  
2. Activate the theme via **Appearance > Themes**.  
3. Install the required/recommended plugins when prompted.  
4. (Optional) Go to **Appearance > Import Demo Data** to load demo content.  
5. Customize your website using Elementor and **Appearance > Customize**.

== Frequently Asked Questions ==  

= Do I need Elementor Pro? =  
No, Brainfwd works perfectly with the free version of Elementor. You can optionally use Elementor Pro for advanced designs.

= Is the theme translation ready? =  
Yes! It includes `.pot` files and supports WPML, Polylang, and Loco Translate.

= Can I sell courses or digital products? =  
Yes. Brainfwd supports WooCommerce, so you can sell courses, books, guides, or any digital product.

= Can I import the demo content? =  
Yes. The theme includes a one-click demo import option to get you started quickly.

= Is this theme only for brain training websites? =  
While it's optimized for brain training and cognitive performance, it can be used for coaching, educational, personal growth, and LMS sites as well.
