== Description ==
Codexse – Elementor Complete Addons is a powerful and complete set of widgets for the Elementor page builder. With a variety of customizable widgets, you can build unique and responsive websites effortlessly.

**Features include:**
* A vast array of custom Elementor widgets
* Gutenberg block support
* Advanced customization for design flexibility
* Regular updates and compatibility improvements

== Installation ==
1. Download the Codexse – Elementor Complete Addons plugin.
2. Upload the `codexse-elementor-addons` folder to your `/wp-content/plugins/` directory, or install the plugin via the WordPress plugins screen.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Access new widgets in Elementor under the "Codexse" category.

== Frequently Asked Questions ==

= Do I need Elementor to use this plugin? =
Yes, this plugin extends the Elementor page builder, so Elementor must be installed and activated.

= Is Codexse – Elementor Complete Addons free? =
Yes, this plugin is completely free to use and is licensed under GPLv2 or later.

= How do I access the widgets? =
After activation, open Elementor, and you’ll find the widgets under the “Codexse” category in the Elementor editor.
